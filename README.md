# 📊 Sales Dashboard & Trend Analyzer

## Project Overview
A complete data analysis project built using Python libraries — NumPy, Pandas, and Matplotlib.
Analyzes 133 sales transactions across 4 product categories, 4 regions, and 12 months.

## 🔧 Technologies Used
- Python 3
- NumPy — Statistical Analysis
- Pandas — Data Manipulation
- Matplotlib — Data Visualization

## 📂 Files
- `Sales_Dashboard_Trend_Analyzer.ipynb` — Main Jupyter Notebook
- `sales.csv` — Sales dataset (133 rows, 9 columns)
- `product_info.csv` — Product details for merge operation

## 📋 Topics Covered
- Loading CSV data with pd.read_csv()
- Handling missing values with fillna()
- Renaming columns and changing data types
- NumPy: mean, std, variance, normalization
- Adding Month, Quarter, Profit_Margin columns using apply() + lambda
- 4 Matplotlib charts: Bar, Line, Pie, Histogram
- GroupBy with multiple columns and .agg()
- Merging DataFrames with pd.merge() inner join
